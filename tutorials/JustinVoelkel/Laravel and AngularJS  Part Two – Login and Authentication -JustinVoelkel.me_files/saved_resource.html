            Gregers Jacobsen Eduardo WB • 2 months ago

            I finally figured it out:
            set ng-model="loginData" on the form-element, and set ng-model="loginData.username" and ng-model="loginData.password" on the username- and password-field respectively. That way you can access the value of the elements as single object using $scope.loginData or the elements individually using $scope.loginData.username or $scope.loginData.password on the elements.

            I hope this helps someone else who finds this otherwise excellent guide :)
